# Structure

The structure of the main page is set up into two sections, the navbar and the main page content. In the following snippet, you can see the actual code behind the first section. The classes are the result of TailwindCSS, the CSS framework used in my website the simplify the design. Tailwinds utility classes let you work with contrains, so you don't overcomplicate your design. More infomation at [TailwindCSS.com](https://tailwindcss.com/)
<customspacer />

<customspacer />
The navbar (or pageHead here) consists of two sections, the `leftDivision` and the `navBar`.
<customspacer />
### `leftDivision`

The left division contains my logo, aswell as the site title in the `<logoText>` element.
<customspacer />
### `navBar`

The navBar element is a wrapper for the four links inside. These are being distributed evenly with the use of a flexbox. They contain an invisible unicode character and a span with the `material-symbols-outlined` class. This span element applies the special Material Symbols font from Google to the text inside. The result looks like this:

<link
	rel="stylesheet"
	href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@48,400,1,0"
/>

		<span class="material-symbols-outlined">description</span>

	</div>
</div>
